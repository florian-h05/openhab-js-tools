(static) dimmer(managerID, targetItem, targetState, step, time, ignoreExternalChangeopt, overwriteopt)

Source:
  • itemutils/dimmer.js, line 30

Controls an Item step-by-step to a target state. Only works for Items with support for float states.

Example
itemutils.dimmer('exampleManager', targetItem, targetState, step, time);
Parameters:
Name Type Attributes Default Description
managerID string

id for the dimmer manager, used as key for the cache
targetItem string

name of the Item to control
targetState number

target to dim to
step number

dimming step size
time number

time in milliseconds between each step
ignoreExternalChange boolean <optional>
 false

whether to break dimmer if Item receives large external change
overwrite boolean <optional>
 false

whether to cancel an existing dimmer and create a new one
Throws:

error when targetItem does not support float state or a parameter has wrong type
