MlscRestClient

thingsx. MlscRestClient

music_led_strip_control REST client

Class providing state fetching from and command sending to the REST API of music_led_strip_control. It is using a scheduled job to fetch states and a rule to handle commands.

Constructor

new MlscRestClient(effectItemName, colorItemName, url, deviceId, switchItemNameopt, effectDefaultopt, refreshIntervalopt)

Description:

  • Be aware that you need to call scheduleStateFetching and createCommandHandlingRule to fully initialize the REST client.

Source:
  • thingsx/mlsc.js, line 30
Example
var FlorianRGB = new MlscRestClient('FlorianRGB_effect', 'FlorianRGB_color', 'http://127.0.0.1:8080', 'device_0', 'FlorianRGB', 'effect_single');
FlorianRGB.scheduleStateFetching();
FlorianRGB.createCommandHandlingRule();
