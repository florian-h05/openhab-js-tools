MlscRestClient

thingsx. MlscRestClient

music_led_strip_control REST client

Class providing state fetching from and command sending to the REST API of music_led_strip_control. It is using a scheduled job to fetch states and a rule to handle commands.

Constructor

new MlscRestClient(config)

Description:

  • Be aware that you need to call scheduleStateFetching and createCommandHandlingRule to fully initialize the REST client.

var { thingsx } = require('@hotzware/openhab-tools');
var mlsc = new thingsx.MlscRestClient({
  effectItemName: 'FlorianRGB_effect',
  url: 'http://127.0.0.1:8080',
  deviceId: 'device_0',

  dimmerItemName: 'FlorianRGB_dimmer'
